# 🚀 Predictive Customer Analytics & Sales Intelligence System

## 📌 Overview

This project is a full-stack AI-powered system that predicts customer behavior and provides business insights using Power BI dashboards.

---

## 🧠 Features

### 🔹 Machine Learning

* Purchase Prediction (Will customer buy again?)
* Churn Prediction (Will customer stop purchasing?)
* Customer Segmentation using K-Means

### 🔹 Backend (Flask APIs)

* `/predict_purchase`
* `/predict_churn`
* `/get_customers`
* `/get_segments`
* `/get_predictions`

### 🔹 Frontend

* Interactive UI for real-time predictions
* Displays results with probability

### 🔹 Power BI Dashboard

* Sales analytics
* Customer segmentation
* Real-time prediction analytics (API-based)

---

## 🏗️ Tech Stack

* **Frontend:** HTML, CSS, JavaScript
* **Backend:** Flask (Python)
* **Database:** SQLite
* **ML Libraries:** scikit-learn, pandas, numpy
* **Visualization:** Power BI

---

## 🔄 System Architecture

Frontend → Flask API → ML Model → Database → Power BI Dashboard

---

## ⚙️ Installation

```bash
git clone https://github.com/nandini-mahadik/customer-analytics-ai-system.git
cd customer-analytics-ai-system
pip install -r requirements.txt
```

---

## ▶️ Run Backend

```bash
cd backend
python app.py
```

---

## 🌐 Run Frontend

Open:

```bash
frontend/index.html
```

## 📊 Power BI

* Open `powerbi/dashboard.pbix`
* Click **Refresh**
* Ensure Flask server is running
